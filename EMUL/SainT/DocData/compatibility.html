Compatibility
Best configuration

SainT requires a cheap PC and a 3d video card to run. It runs at 50Hz on a classic Pentium 2, 233Mhz with a voodoo 1 card. Faster machine should run it fine too !

Please note you need to install Microsoft DirectX 8.1 Runtime. ( Free download available at Microsoft website )

If you don't want problem, the best configuration is: (see the Option page)

  • TOS 1.00
  • STF machine (not STE)
  • CPU rendering, normal mode

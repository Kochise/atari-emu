Directory To MSA
Written by Arnaud Carré

* Now support ZIP file directly *

 

SainT emulator actually only support floppy disk image file. If you download some nice file-based ATARI demos, you need to xform these files to a floppy disk image. That's exactly what DIR2MSA tool do !

User guide

Put all files you need in a directory of your choice. That directory will be the root of the future floppy disk. And simply drag-and-drop the directory to the Dir2Msa.exe tool, and Voila ! A MSA file (<directory name>.msa) will be created at the same level of directoy. You can use that MSA disk image with SainT or any ATARI-ST emulator.

Technical

I fastly write that tool which play with 12bits FAT system, so you may experience some bad image conversion (I never have). If you get errors please mail a ZIP file of the directory you try to convert.

Have fun !

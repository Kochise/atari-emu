Hatari for Windows was built using the mingw64 version of the gcc compiler,
using the following libraries (those dll were also built with mingw64-gcc) :

SDL2.dll		version 2.0.12
http://libsdl.org/download-2.0.php

libpng16-16		version 1.6.37
http://sourceforge.net/projects/libpng/

readline.dll		version 8.0
http://www.gnu.org/s/readline/

libtermcap-0.dll	version 1.3.1
http://ftp.gnu.org/gnu/termcap/

zlib1.dll		version 1.2.11
http://zlib.net/

capsimg.dll		version 5.1
http://softpres.org/download/
